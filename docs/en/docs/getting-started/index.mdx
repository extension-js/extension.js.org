# Getting started with Extension.js

Ship your first browser extension quickly, then level up with templates and workflow guides as your project grows.

## Who this path is for

- **New to extension development:** [Create your first extension](./create-your-first-extension.mdx)
- **Exploring quickly with existing samples:** [Get started immediately](./immediately.mdx)
- **Choosing a starting stack:** [Templates](./templates.mdx)

## First successful outcome

1. Scaffold or run an extension project.
2. Launch `dev` and load the extension in a browser target.
3. Make one change and verify update behavior.
4. Build production output for at least one browser.
