## Create the project

<CodeGroup>

```bash npm
npx extension@latest create my-extension --template=react
```

```bash pnpm
pnpx extension@latest create my-extension --template=react
```

```bash yarn
yarn dlx extension@latest create my-extension --template=react
```

</CodeGroup>

## What you get

A component-based UI baseline with full Manifest V3 (MV3) compatibility. Popup, options, and content scripts ship as React entry points with HMR wired up, so you can edit a component and see it update without manually reloading the extension.

The most common starting point for teams already comfortable with React.
