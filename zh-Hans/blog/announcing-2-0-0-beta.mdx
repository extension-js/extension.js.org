<Update label="v2.0.0-beta" description="October 18, 2024" tags={["release", "beta"]}>

![Extension.js 2.0.0-beta banner](/images/blog/extension-2-0-0-beta.png)

Extension.js 现已正式进入 beta 阶段。

本次 beta 版本稳定了 alpha 阶段引入的特性，并修复了所有已知的浏览器兼容性问题。

**版本 2.0.0-beta**

- 修复 Sass Modules 在 content script 中无法工作的问题。
- 新增对 Svelte 模板的支持。
- 新增 preview 命令。
- 为 content_scripts 提供通用支持（通过硬重载与热重载）。
- 修复 Firefox、Edge 与 Chrome 上所有已知的浏览器问题。
- 更新文档（就是这一份！）。

Beta 版本聚焦于让所有现有特性达到可文档化、可测试的稳定程度。在下一个稳定版本之前，我们已经冻结新功能。文档目前已经是最新版。这将是发布候选版之前最后一个 beta 版本。

下一个版本，我们计划让 Extension.js 拥有更快、更小的包体。它将同时支持本地与远程运行（通过 `pnpm dlx`、`yarn dlx` 和 `npx`）。

感谢阅读。再见。

Cezar Augusto<br/>
Extension.js 创建者与主开发者

</Update>
