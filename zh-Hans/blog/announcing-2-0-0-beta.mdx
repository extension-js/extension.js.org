<Update label="v2.0.0-beta" description="October 18, 2024" tags={["release", "beta"]}>

![Extension.js 2.0.0-beta banner](/images/blog/extension-2-0-0-beta.png)

Extension.js 现在正式进入 beta 阶段。

此 beta 版稳定了 alpha 版本中引入的特性，并修复了所有已知的浏览器兼容性问题。

**版本 2.0.0-beta**

- 修复 Sass Modules 在内容脚本中无法工作的问题。
- 增加对 Svelte 模板的支持。
- 新增 preview 命令。
- 通过 hard-reload 与 hot-reload 为 content_scripts 提供通用支持。
- 修复 Firefox、Edge 与 Chrome 上所有已知的浏览器问题。
- 更新文档（就是这个！）

beta 版本专注于让所有现有功能足够稳定，从而能够完成文档化和测试。在下一个稳定版发布之前，我们已经锁定了新特性。文档现已更新。这将是发布候选版本（RC）之前的最后一个 beta 版本。

在下一个版本中，我们计划让 Extension.js 拥有更快、更小的产物。它将既能在本地工作，也能远程运行（通过 `pnpm dlx`、`yarn dlx` 和 `npx`）。

感谢阅读。回头聊。
