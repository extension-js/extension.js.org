<Update label="v2.0.0-beta" description="October 18, 2024" tags={["release", "beta"]}>

![Extension.js 2.0.0-beta banner](/images/blog/extension-2-0-0-beta.png)

Extension.js 现在正式进入 beta 阶段。

这次 beta 让 alpha 中引入的特性更加稳定，并修复了所有已知的浏览器兼容性问题。

**Version 2.0.0-beta**

- 修复 Sass Modules 在 content script 中无法工作的问题。
- 加入对 Svelte 模板的支持。
- 新增 preview 命令。
- 对 content_scripts 提供通用支持（通过硬重载与热重载）。
- 修复 Firefox、Edge 和 Chrome 上所有已知的浏览器问题。
- 更新文档（就是你正在看的这个！）。

这次 beta 主要聚焦在让所有现有特性达到足够稳定、可文档化和可测试的状态。在下一个稳定版之前，我们冻结了新特性。文档已经更新完毕。这是发布候选版之前的最后一个 beta。

下个版本，我们计划让 Extension.js 的产物更快、体积更小。它将同时在本地和远程工作（通过 `pnpm dlx`、`yarn dlx` 和 `npx`）。

感谢阅读，回头再聊。

Cezar Augusto<br/>
Creator and Lead Developer, Extension.js
