## 第一次成功的目标

<Steps>
  <Step title="搭建或运行一个扩展项目">
    使用 `extension create` 或选择一个模板作为起点。
  </Step>
  <Step title="启动 dev 并加载扩展">
    运行 `extension dev`,并将构建产物加载到目标浏览器中。
  </Step>
  <Step title="验证更新行为">
    做一个修改,确认扩展会自动重载。
  </Step>
  <Step title="构建生产环境产物">
    至少为一种浏览器运行 `extension build`。
  </Step>
</Steps>
