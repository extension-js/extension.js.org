## 第一次成功的目标

<Steps>
  <Step title="搭建或运行一个扩展项目">
    使用 `extension create`，或选择一个模板作为起点。
  </Step>
  <Step title="启动 dev 并加载扩展">
    运行 `extension dev`，把构建产物加载到目标浏览器中。
  </Step>
  <Step title="验证更新行为">
    修改一处，确认扩展能够重新加载。
  </Step>
  <Step title="构建生产产物">
    至少为一个浏览器运行 `extension build`。
  </Step>
</Steps>
