## 首个成功结果

<Steps>
  <Step title="脚手架或运行一个扩展项目">
    使用 `extension create` 或选择一个模板作为起点。
  </Step>
  <Step title="启动 dev 并加载扩展">
    运行 `extension dev` 并在目标浏览器中加载构建输出。
  </Step>
  <Step title="验证更新行为">
    做一次修改,确认扩展会重新加载。
  </Step>
  <Step title="构建生产输出">
    至少为一个浏览器运行 `extension build`。
  </Step>
</Steps>
