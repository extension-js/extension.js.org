Extension.js 会从项目依赖中检测 React，并自动配置 JSX/TSX 转换、React 模块别名，以及开发时的 fast refresh 支持。

## 适合使用 React 的场景

- 你正在构建丰富的 UI 界面，例如弹窗、选项页、侧边栏或新标签页。
- 你需要在扩展页面和内容脚本 UI 挂载点之间复用组件。
- 你已经在 Web 项目中使用 React，并希望在扩展中使用相同的开发模式。

## 在现有扩展中使用

按以下步骤为现有扩展添加 React 支持。

### 安装

安装 React 运行时依赖：

<PackageManagerTabs command="install react react-dom" />

如果你使用 TypeScript，也请安装 React 类型包：

<PackageManagerTabs command="install -D @types/react @types/react-dom" />

### 配置

常见的 React 文件模式：

- JavaScript：`*.jsx`
- TypeScript：`*.tsx`

## 开发行为

在开发模式下，Extension.js 检测到 React 时会启用 React refresh 集成。

- Extension.js 使用 `react-refresh` 和 `@rspack/plugin-react-refresh` 实现 refresh 行为。
- 如果项目缺少可选的 refresh 依赖，Extension.js 会自动安装并提示你重启。
- Extension.js 会应用 React 别名，以保持打包中只有一个 React/runtime 实例。

## 用法示例

### 在新标签页扩展中

页面入口示例：

```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    <title>New Extension</title>
  </head>
  <body>
    <noscript>You need to enable JavaScript to run this extension.</noscript>
    <div id="root"></div>
  </body>
  <script src="./index.tsx"></script>
</html>
```

```tsx
import React from "react";
import ReactDOM from "react-dom/client";
import App from "./App";

const root = ReactDOM.createRoot(document.getElementById("root"));

root.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
);
```

```tsx
export default function App() {
  return <h1>Hello, Extension!</h1>;
}
```

### 在 `content_script` 文件中

对于内容脚本，将 React 渲染到注入的根元素中：

```tsx
import React from "react";
import ReactDOM from "react-dom/client";
import App from "./App";
import "./content.css";

const rootDiv = document.createElement("div");
rootDiv.id = "extension-root";
document.body.appendChild(rootDiv);

const root = ReactDOM.createRoot(rootDiv);
root.render(<App />);
```
