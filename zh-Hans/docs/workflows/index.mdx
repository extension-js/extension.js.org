## 发布信心闭环

1. 运行有针对性的本地检查（`dev` + 故障排查）。
2. 构建生产产物（`build` / `start`）。
3. 用端到端 (E2E) 与冒烟检查验证关键流程。
4. 在应用商店提交前运行 CI 模板。
