## 推荐学习路径

1. 校验浏览器产物保证(`cross-browser-compatibility`)。
2. 锁定项目默认值(`extension-configuration`)。
3. 加入进阶行为(`rspack-configuration`、`environment-variables`)。
4. 优化发布流程(`multi-platform-builds`)。
