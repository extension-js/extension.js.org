## 推荐进度

1. 校验浏览器输出保障(`cross-browser-compatibility`)。
2. 锁定项目默认值(`extension-configuration`)。
3. 添加高级行为(`rspack-configuration`、`environment-variables`)。
4. 优化发布流程(`multi-platform-builds`)。
