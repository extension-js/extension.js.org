## 推荐顺序

1. 验证浏览器输出保证(`cross-browser-compatibility`)。
2. 锁定项目默认值(`extension-configuration`)。
3. 添加进阶行为(`rspack-configuration`、`environment-variables`)。
4. 优化发布流程(`multi-platform-builds`)。
