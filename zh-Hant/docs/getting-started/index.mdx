## 第一次成功的成果

<Steps>
  <Step title="建立或執行一個擴充功能專案">
    使用 `extension create` 或選一個範本作為起點。
  </Step>
  <Step title="啟動 dev 並載入擴充功能">
    執行 `extension dev`，並在目標瀏覽器中載入建置輸出。
  </Step>
  <Step title="驗證更新行為">
    做一次變更，並確認擴充功能會自動重新載入。
  </Step>
  <Step title="建置正式輸出">
    至少對一種瀏覽器執行 `extension build`。
  </Step>
</Steps>
