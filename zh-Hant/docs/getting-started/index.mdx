## 第一次成功的目標

<Steps>
  <Step title="搭建或執行一個擴充功能專案">
    使用 `extension create` 或選擇一個範本作為起點。
  </Step>
  <Step title="啟動 dev 並載入擴充功能">
    執行 `extension dev`,並將建置產物載入到目標瀏覽器中。
  </Step>
  <Step title="驗證更新行為">
    做一次修改,確認擴充功能會重新載入。
  </Step>
  <Step title="建置正式環境產物">
    至少對一種瀏覽器執行 `extension build`。
  </Step>
</Steps>
