## 第一個成功成果

<Steps>
  <Step title="建立或執行一個擴充功能專案">
    使用 `extension create` 或選擇一個範本作為起點。
  </Step>
  <Step title="啟動 dev 並載入擴充功能">
    執行 `extension dev` 並將建構輸出載入到瀏覽器目標中。
  </Step>
  <Step title="驗證更新行為">
    做一個變更並確認擴充功能會重新載入。
  </Step>
  <Step title="建構正式版輸出">
    為至少一個瀏覽器執行 `extension build`。
  </Step>
</Steps>
