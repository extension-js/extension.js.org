## 第一個成功結果

<Steps>
  <Step title="建立或執行一個擴充功能專案">
    使用 `extension create`，或挑一個範本作為起點。
  </Step>
  <Step title="啟動 dev 並載入擴充功能">
    執行 `extension dev`，並在指定的瀏覽器目標中載入建置輸出。
  </Step>
  <Step title="驗證更新行為">
    修改一處內容,確認擴充功能會重新載入。
  </Step>
  <Step title="建置正式版輸出">
    至少對一個瀏覽器執行 `extension build`。
  </Step>
</Steps>
