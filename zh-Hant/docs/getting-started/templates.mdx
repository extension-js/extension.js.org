透過從官方範本搭建擴充功能專案,不必從零打造資料夾結構和設定,讓你能更快開始。

## 範本能力

- **現成的專案結構:** 提供可運作的 manifest、腳本和原始碼配置。
- **針對技術堆疊的預設:** 依框架和執行時需求挑選範本。
- **更快上手:** 幾分鐘內從想法到可執行的擴充功能。
- **一致的預設值:** 減少專案與團隊成員之間的設定漂移。

## 快速使用

<CodeGroup>

```bash npm
npx extension@latest create my-extension --template=<slug>
```

```bash pnpm
pnpx extension@latest create my-extension --template=<slug>
```

```bash yarn
yarn dlx extension@latest create my-extension --template=<slug>
```

```bash bun
bunx extension@latest create my-extension --template=<slug>
```

```bash bun
bunx extension@latest create my-extension --template=<slug>
```

</CodeGroup>

將 `<slug>` 替換為你想使用的範本。

## 運作原理

當你執行 `create --template=<slug>` 時,Extension.js 會抓取所選範本、產生專案檔案,並選擇性地安裝相依套件。
