了解使 Extension.js 在實際瀏覽器擴充功能開發中切實可用的核心平台能力。

| 目標                                           | 從這裡開始                                                                    |
| ---------------------------------------------- | ----------------------------------------------------------------------------- |

| 妥善組織非 manifest 資源                       | [特殊資料夾](/docs/features/special-folders)                                  |
| 管理環境特定行為                               | [環境變數](/docs/features/environment-variables)                              |
| 集中管理建構/執行階段預設值                    | [Extension 設定](/docs/features/extension-configuration)                      |
| 安全地擴充打包器行為                           | [Rspack 設定](/docs/features/rspack-configuration)                            |

## 建議的學習進程

1. 驗證瀏覽器輸出保證（`cross-browser-compatibility`）。
2. 鎖定專案預設值（`extension-configuration`）。
3. 加入進階行為（`rspack-configuration`、`environment-variables`）。
4. 最佳化發行流程（`multi-platform-builds`）。
