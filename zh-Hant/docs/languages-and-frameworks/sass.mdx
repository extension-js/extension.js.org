利用 Sass 透過變數、巢狀規則與 SCSS modules 擴充瀏覽器擴充功能樣式，同時維持單一的建置工作流程。

Extension.js 透過以 Rspack 為基礎的樣式流水線，支援 `.scss` 與 `.sass`，以及 SCSS module 變體（`.module.scss`、`.module.sass`）。

## 何時適合使用 Sass

- 你的設計系統已經依賴 Sass 變數與 mixin。
- 你需要為元件導向的擴充功能 UI 提供模組化樣式。
- 你希望與既有的 Sass 網頁應用工作流程保持一致。

## 範本範例

### `new-sass`

![new-sass template screenshot](/images/examples/new-sass/screenshot.png)

以已設定好的 Sass 支援，建立新分頁擴充功能。

<CodeGroup>

```bash npm
npx extension@latest create my-extension --template=new-sass
```

```bash pnpm
pnpx extension@latest create my-extension --template=new-sass
```

```bash yarn
yarn dlx extension@latest create my-extension --template=new-sass
```

```bash bun
bunx extension@latest create my-extension --template=new-sass
```

```bash bun
bunx extension@latest create my-extension --template=new-sass
```

</CodeGroup>

import "./styles/globals.scss";

## Sass modules

像 CSS modules 一樣，使用 Sass modules 將樣式限定在區域作用域。使用 `.module.scss` 或 `.module.sass` 啟用作用域，並避免樣式表中的命名衝突。

## 在既有擴充功能中使用 Sass modules

要啟用 Sass modules，請將 Sass 檔案重新命名為含 `.module.scss` 或 `.module.sass`。這會自動將樣式限定到你的元件作用域。

```diff
- button.scss
+ button.module.scss
```

### 使用範例

重新命名 Sass 檔案後，將它匯入你的腳本：

```ts
import styles from "./styles/button.module.scss";

const element = document.createElement("h1");
element.className = styles.primaryColor;
element.innerText = "Hello, Extension!";
document.body.appendChild(element);
```

在 Sass 檔案中：

```scss
/* styles/button.module.scss */

$primary-color: #64ff00;

:export {
  primaryColor: $primary-color;
}
```

## React／Preact 使用範例

將 Sass module 匯入你的 React 或 Preact 元件，並使用作用域類別名稱：

```jsx
import styles from "./styles/button.module.scss";

export default function MyComponent() {
  return <h1 className={styles.primaryColor}>Hello, Extension!</h1>;
}
```

## Vue 使用範例

在 Vue SFC 中，建議使用 `lang="scss"` 的模組樣式：

```vue
<template>
  <h1 :class="$style.primaryColor">Hello, Extension!</h1>
</template>

<style lang="scss" module>
@import "./styles/button.module.scss";
</style>
```

## Svelte 使用範例

在 Svelte 中，匯入 Sass 模組對應並套用類別：

```svelte
<script>
  import styles from "./styles/button.module.scss";
</script>

<h1 class={styles.primaryColor}>Hello, Extension!</h1>
```

## 行為注意事項

- 在擴充功能頁面中，`.module.scss` 與 `.module.sass` 會匯出類別對應供 JS／TS 匯入。
- 在內容腳本中，Extension.js 會將 Sass 輸出為 CSS 資產，不附加 JavaScript 類別名稱對應。
- 若你的專案缺乏 Sass 工具，Extension.js 會安裝選用相依套件，並要求你重新啟動。
