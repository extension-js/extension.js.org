Extension.js 會從你的專案相依套件偵測 React，並自動設定 JSX／TSX 轉換、React 模組別名，以及開發階段的 fast-refresh 支援。

## 何時適合使用 React

- 你正在建構像 popup、options、sidebar 或新分頁等豐富的 UI 介面。
- 你希望在擴充功能頁面與內容腳本 UI 掛載點之間重複使用元件。
- 你已在網頁專案中使用 React，希望在擴充功能中沿用相同的開發模型。

## 在既有擴充功能中使用

依下列步驟，將 React 加入既有擴充功能。

### 安裝

安裝 React 執行階段相依套件：

<PackageManagerTabs command="install react react-dom" />

如果你使用 TypeScript，也請安裝 React 型別套件：

<PackageManagerTabs command="install -D @types/react @types/react-dom" />

### 設定

常見的 React 檔案模式：

- JavaScript：`*.jsx`
- TypeScript：`*.tsx`

## 開發行為

在開發模式下，當 Extension.js 偵測到 React 時，會啟用 React refresh 整合。

- Extension.js 使用 `react-refresh` 與 `@rspack/plugin-react-refresh` 處理 refresh 行為。
- 若你的專案缺少選用 refresh 相依套件，Extension.js 會安裝它們並要求你重新啟動。
- Extension.js 套用 React 別名，以在套件中維持單一 React／執行階段實例。

## 使用範例

### 在新分頁擴充功能中

頁面進入點範例：

```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    <title>New Extension</title>
  </head>
  <body>
    <noscript>You need to enable JavaScript to run this extension.</noscript>
    <div id="root"></div>
  </body>
  <script src="./index.tsx"></script>
</html>
```

```tsx
import React from "react";
import ReactDOM from "react-dom/client";
import App from "./App";

const root = ReactDOM.createRoot(document.getElementById("root"));

root.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
);
```

```tsx
export default function App() {
  return <h1>Hello, Extension!</h1>;
}
```

### 在 `content_script` 檔案中

對於內容腳本，請將 React 渲染至注入的根元素：

```tsx
import React from "react";
import ReactDOM from "react-dom/client";
import App from "./App";
import "./content.css";

const rootDiv = document.createElement("div");
rootDiv.id = "extension-root";
document.body.appendChild(rootDiv);

const root = ReactDOM.createRoot(rootDiv);
root.render(<App />);
```
