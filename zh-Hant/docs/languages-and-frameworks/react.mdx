Extension.js 會從你的專案相依套件中偵測到 React,並自動為開發環境設定 JSX/TSX 轉換、React 模組別名以及 fast-refresh 支援。

## 何時適合使用 React

- 你正在打造豐富的 UI 介面,如彈出視窗、選項頁、側邊欄或新分頁。
- 你希望在擴充功能頁面與內容腳本 UI 掛載之間重複使用元件。
- 你已在 Web 專案中使用 React,並希望在擴充功能中獲得相同的開發模式。

## 在現有擴充功能中使用

依照以下步驟將 React 加入既有的擴充功能。

### 安裝

安裝 React 執行時相依套件:

<PackageManagerTabs command="install react react-dom" />

若你使用 TypeScript,請一併安裝 React 型別套件:

<PackageManagerTabs command="install -D @types/react @types/react-dom" />

### 設定

常見的 React 檔案模式:

- JavaScript:`*.jsx`
- TypeScript:`*.tsx`

## 開發行為

在開發模式下,當 Extension.js 偵測到 React 時,會啟用 React refresh 整合。

- Extension.js 透過 `react-refresh` 與 `@rspack/plugin-react-refresh` 提供 refresh 行為。
- 若你的專案缺少選用的 refresh 相依套件,Extension.js 會自動安裝並要求重新啟動。
- Extension.js 會套用 React 別名,確保產物中維持單一的 React/執行時實例。

## 使用範例

### 在新分頁擴充功能中

範例頁面進入點:

```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    <title>New Extension</title>
  </head>
  <body>
    <noscript>You need to enable JavaScript to run this extension.</noscript>
    <div id="root"></div>
  </body>
  <script src="./index.tsx"></script>
</html>
```

```tsx
import React from "react";
import ReactDOM from "react-dom/client";
import App from "./App";

const root = ReactDOM.createRoot(document.getElementById("root"));

root.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
);
```

```tsx
export default function App() {
  return <h1>Hello, Extension!</h1>;
}
```

### 在 `content_script` 檔案中

針對內容腳本,將 React 渲染到注入的根元素中:

```tsx
import React from "react";
import ReactDOM from "react-dom/client";
import App from "./App";
import "./content.css";

const rootDiv = document.createElement("div");
rootDiv.id = "extension-root";
document.body.appendChild(rootDiv);

const root = ReactDOM.createRoot(rootDiv);
root.render(<App />);
```
