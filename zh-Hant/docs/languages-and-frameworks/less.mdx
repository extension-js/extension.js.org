透過一套同時支援模組範圍樣式的統一流程,在瀏覽器擴充功能頁面與 UI 元件中使用 Less CSS。

Extension.js 會偵測 Less 的使用情況,在 Rspack 建置中設定前處理器,並在瀏覽器擴充功能中同時支援 `.less` 與 `.module.less` 檔案。

## 何時適合使用 Less

- 你正在遷移既有的基於 Less 的 Web 程式碼庫。
- 你希望以較低的遷移成本取得變數與巢狀語法。
- 你需要模組範圍樣式,同時保留 Less 的撰寫慣例。

## 在現有擴充功能中使用

依照以下步驟將 Less 加入既有的擴充功能。

### 安裝

安裝所需相依套件:

<PackageManagerTabs command="install -D less less-loader" />

### 在 HTML 檔案中的使用範例

在擴充功能頁面(彈出視窗/選項頁/新分頁)中,透過腳本進入點匯入 Less:

```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    <title>New Extension</title>
    <script src="./main.ts"></script>
  </head>
  <body>
    <h1 class="title">Hello, Extension.</h1>
  </body>
</html>
```

```ts
import "./styles/globals.less";

console.log("Less loaded");
```

### 在 `content_script` 檔案中

```ts
import "./styles/content.less";

console.log("Content styles loaded");
```

## 瀏覽器擴充功能中的 Less modules

如同 CSS modules,使用 Less modules 可在本地為類別名稱建立範圍,預設避免全域樣式衝突。這在擴充功能中尤其重要,因為內容腳本會在宿主頁面的 CSS 範圍內執行。

### `content-less-modules`

![content-less-modules 範本截圖](/images/examples/content-less-modules/screenshot.png)

當你需要在注入的 UI 中使用範圍化樣式時,在內容腳本中使用 Less modules。

<CodeGroup>

```bash npm
npx extension@latest create my-extension --template=content-less-modules
```


## 在現有擴充功能中使用 Less modules

要啟用 Less modules,請將 `.less` 檔案重新命名,加上 `.module.less` 後綴。這會自動為類別名稱啟用本地範圍。

```diff
- myStyles.less
+ myStyles.module.less
```

### 使用範例

重新命名 Less 檔案後,可以在腳本中匯入它:

```ts
import styles from "./styles/myStyles.module.less";

const element = document.createElement("h1");
element.className = styles.primary;
element.innerText = "Hello, Extension!";
document.body.appendChild(element);
```

## React/Preact 使用範例

若要在 React 或 Preact 中使用 Less modules,匯入 Less module 後便可像在任何元件中一樣套用樣式:

```jsx
import styles from "./styles/myStyles.module.less";

export default function MyComponent() {
  return <h1 className={styles.primary}>Hello, Extension!</h1>;
}
```

## Vue 使用範例

在 Vue 中,建議使用搭配 `lang="less"` 與 `module` 的單一檔案元件(SFC)樣式:

```vue
<template>
  <h1 :class="$style.primary">Hello, Extension!</h1>
</template>

<style lang="less" module>
@import "./styles/myStyles.module.less";
</style>
```

## Svelte 使用範例

在 Svelte 中,匯入 Less module 的映射並套用類別名稱:

```svelte
<script>
  import styles from "./styles/myStyles.module.less";
</script>

<h1 class={styles.primary}>Hello, Extension!</h1>
```

## 行為說明

- 在擴充功能頁面情境中,`.module.less` 會如預期匯出類別名稱映射。
- 在內容腳本中,Extension.js 會將樣式作為 CSS 資源輸出,並以樣式表方式注入頁面。
- 若你的專案缺少 Less 工具,Extension.js 會安裝選用相依套件並要求重新啟動。
