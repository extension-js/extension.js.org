在瀏覽器擴充功能頁面與 UI 元件中使用 Less CSS,並透過同一條管線同時支援 module 範圍隔離的樣式。

Extension.js 會偵測 Less 的使用情境、在 Rspack 建置中配置預處理器,並在瀏覽器擴充功能中同時支援 `.less` 與 `.module.less` 檔案。

## 何時適合使用 Less

- 你正在遷移既有以 Less 為基礎的 Web 程式庫。
- 你想以低遷移成本擁有變數與巢狀語法。
- 你需要 module 範圍的樣式,同時保留 Less 慣例。

### 在 HTML 檔中的使用範例

在擴充功能頁面(popup、options、新分頁)中,透過你的腳本進入點匯入 Less:

```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    <title>New Extension</title>
    <script src="./main.ts"></script>
  </head>
  <body>
    <h1 class="title">Hello, Extension.</h1>
  </body>
</html>
```

```ts
import "./styles/globals.less";

console.log("Less loaded");
```

### 在 `content_script` 檔案中

```ts
import "./styles/content.less";

```

## 瀏覽器擴充功能中的 Less modules

跟 CSS modules 一樣,可以用 Less modules 將類別名稱範圍區域化。預設能避免全域樣式衝突,這對擴充功能尤其重要,因為 content script 會在主頁面的 CSS 範圍中執行。

### `content-less-modules`

![content-less-modules template screenshot](/images/examples/content-less-modules/screenshot.png)

當你需要為注入的 UI 套用範圍隔離的樣式時,在 content script 中使用 Less modules。

## 在既有擴充功能中使用 Less modules

要啟用 Less modules,請將你的 `.less` 檔重新命名,加上 `.module.less`。如此一來,類別名稱會自動套用區域範圍。

```diff
- myStyles.less
+ myStyles.module.less
```

### 使用範例

重新命名 Less 檔後,即可在腳本中匯入它:

```ts
import styles from "./styles/myStyles.module.less";

const element = document.createElement("h1");
element.className = styles.primary;
element.innerText = "Hello, Extension!";
document.body.appendChild(element);
```

## React/Preact 使用範例

要在 React 或 Preact 中使用 Less modules,只要匯入 Less module,並像在任何元件中一樣套用樣式即可:

```jsx
import styles from "./styles/myStyles.module.less";

export default function MyComponent() {
  return <h1 className={styles.primary}>Hello, Extension!</h1>;
}
```

## Vue 使用範例

在 Vue 中,優先使用單一檔案元件(SFC)的樣式區塊,並指定 `lang="less"` 與 `module`:

```vue
<template>
  <h1 :class="$style.primary">Hello, Extension!</h1>
</template>

<style lang="less" module>
@import "./styles/myStyles.module.less";
</style>
```

## Svelte 使用範例

在 Svelte 中,匯入 Less module 對應並套用類別:

```svelte
<script>
  import styles from "./styles/myStyles.module.less";
</script>

<h1 class={styles.primary}>Hello, Extension!</h1>
```

## 行為注意事項

- 在擴充功能頁面環境中,`.module.less` 會如預期匯出類別對應 map。
- 在 content script 中,Extension.js 會將樣式輸出為 CSS 資源,並以樣式表的方式注入頁面。
- 若你的專案缺少 Less 工具,Extension.js 會安裝選用的相依套件並要求重新啟動。
