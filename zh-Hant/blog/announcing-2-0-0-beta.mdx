<Update label="v2.0.0-beta" description="October 18, 2024" tags={["release", "beta"]}>

![Extension.js 2.0.0-beta banner](/images/blog/extension-2-0-0-beta.png)

Extension.js 現在正式進入 beta 階段。

這個 beta 版穩定了 alpha 版本所引入的功能，並修正所有已知的瀏覽器相容性問題。

**2.0.0-beta 版本**

- 修正 Sass Modules 在內容腳本中無法運作的問題。
- 新增 Svelte 範本支援。
- 新增 preview 命令。
- 為 content_scripts 提供通用支援（透過 hard-reload 與 hot-reload）。
- 修正 Firefox、Edge 與 Chrome 所有已知的瀏覽器問題。
- 更新文件（就是你看到的這個！）

beta 版本聚焦於讓所有既有功能達到足以撰寫文件並進行測試的穩定程度。在下一個穩定版本之前，我們已凍結新功能。文件現在已是最新狀態。這是 release candidate 前的最後一個 beta 版本。

下一個版本，我們計畫讓 Extension.js 成為更快、更小的打包檔。它將能在本地與遠端執行（透過 `pnpm dlx`、`yarn dlx` 與 `npx`）。

感謝你閱讀。再聊。
