<Update label="v2.0.0-alpha" description="September 9, 2024" tags={["release", "alpha"]}>

![Extension.js 2.0.0-alpha banner](/images/blog/extension-2-0-0-alpha.png)

Extension.js 的目標就是讓跨瀏覽器擴充功能開發盡可能簡單。

自從[登上 HackerNews](https://hnrankings.info/40210627/) 以來，我們做了許多內部改進，讓套件更小、更快、對開發者更友善。

對我來說，「非常簡單」代表認知負擔極低，代表事物是熟悉的。開發者最熟悉的就是複製貼上。身為一名開發者，如果我能複製貼上程式碼，並在不做任何設定的情況下立刻看到視覺回饋，這就感覺非常簡單。即使一開始不理解程式碼，快速的視覺回饋就是事物**預期**運作方式最清楚的指標。這就是 Extension.js v2 的期望：非常容易使用。

**Version 2.0.0-alpha**

- 為熱門 JavaScript 框架、CSS 預處理器與擴充功能 API 提供 30 多個範本。[在 GitHub 上查看](https://github.com/extension-js/examples/tree/main/examples)。
- 支援在 manifest.json 中加入只套用於目標瀏覽器的瀏覽器專屬欄位，例如 `{chrome:service_worker: "sw.js"}`
- `extension.config.js` — 適用於需要存取編譯流程的進階情境。[參閱設定文件](/zh-Hant/docs/features/extension-configuration)。
- 採用 [SWC（Speedy Web Compiler）](https://swc.rs/)讓編譯更快速。
- 全新改版的文件網站（就是這個！）。
- 全新範本網站（即將推出）。

Alpha 版本聚焦於錯誤修正並測試所有既有功能，同時我們正在完成文件與範本網站。請繼續關注本部落格，或在 X 上追蹤我們以掌握未來更新。

下一個版本，我們會專注讓所有既有功能穩定到足以撰寫文件與測試。

感謝你的閱讀，下次見。
